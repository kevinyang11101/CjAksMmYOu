# Python学生管理系统 python463

本项目是一款基于Python开发的学生管理系统，采用Django作为后端框架，结合MySQL进行数据存储，前端使用Vue.js构建用户界面。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

系统包含以下主要功能模块：

- **登录注册**：支持用户的登录与注册功能
- **学生管理**：对学生信息进行增删改查管理
- **课程管理**：对课程信息进行维护管理
- **学生成绩管理**：管理学生的成绩信息
- **选课信息管理**：管理学生的选课信息
- **课程类型管理**：对课程类型进行分类管理
- **公告管理**：发布与更新校园公告
- **教师管理**：对教师信息进行管理
- **个人中心**：用户可查看与编辑个人信息

## 系统角色

- 管理员：负责系统的维护与管理
- 教师：负责课程教学、成绩录入等
- 学生：选课、查看成绩、个人信息管理等

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- Vue.js环境

## 部署步骤

1. 环境准备：安装Python、MySQL、Node.js等必要环境
2. 数据库搭建：根据项目需求创建数据库和表结构
3. 后端部署：使用Django框架搭建后端服务
4. 前端部署：使用Vue.js搭建前端界面
5. 系统整合与测试
